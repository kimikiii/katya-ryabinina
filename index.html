Файлы с расширением. html открываются в браузере и состоят из тегов.

  • Форматирования текста
  • Выделения текста
  • Графических элементов
  • Структура страницы

Справочник тегов.

1. Теги для форматирования текста.

  • p - абазац от англ. paragraph
  • h1..h6 - загаловки от англ. heading.
  • ul/ol - маркированный и нумерованный спики от англ. unorder/ordered list, li - пункты списков от англ. list item подробнее
  • a - якорь от англ. anchor

2. Теги для выделения текста.

  • b - жирный от англ. bold
  • i - курсивный от англ. italic
  • del - перечеркнутый от англ. delete

3. Теги для графических элементов.

  • hr - линия от англ. horizontal rule
  • img - картинка от англ. image

4. Теги задающие структуру страницы.

  • !DOCTYPE html - не тег, а инструкция для браузера
  • html - заключает в себе всё содержимое веб-страницы
  • head - для хранения служебной информации о странице
  • body - содержимое страницы
  • header - шапка сайта
  • nav - меню сайта
  • main - главный раздел сайта
  • footer - подвал сайта
  • section - задает раздел документа
  • div - универсальный контейнер (блок)
